FliSdkDemo Python instructions:
	- download the last version of python https://www.python.org/downloads/
	- install PyQt5: pip install PyQt5 (tested with version 5.13.2)
	- install Pillow: pip install Pillow
	- install numpy: pip install numpy
	- install matplotlib: pip install matplotlib
	- start the sofware : "python ./FLiSdkDemo.py"
